## Pre-Existing Failures

### 1. Rust Formatting Drift
- **Issue**: `cargo fmt --check` fails when run in each crate directory
- **Affected files**:
  - `rust/library` (long `read_json(...)` call not wrapped)
  - `rust/component/src/bindings.rs` (generated by wit-bindgen; `link_section` attribute wrapping)
  - `rust/component/tests/wasm_count_tasks.rs` (unsorted `use wasmtime_wasi::{...}` imports)
- **Note**: `cargo clippy --all-targets -- -D warnings` passes clean in both crates.
  There is intentionally no cargo workspace at `rust/` root; per-crate invocation
  is the correct baseline method. Fix scheduled for Phase 9.

- **Affected**: Both `javascript/component` and `javascript/library`
- **Severity**: High (1 high per package)
- **Mitigation available**: `npm audit fix` suggested
